# 🗺️ 위치기반 휴지통/공중화장실 지도 서비스

주소를 입력하면 가까운 휴지통 또는 공중화장실 위치를 지도에 시각화해주는 Streamlit 웹앱입니다.

---

## 📁 프로젝트 폴더 구조

| 경로/파일명 | 설명 |
|-------------|------|
| `app.py` | 🖥️ Streamlit 메인 웹앱 코드 |
| `data_loader.py` | 📦 데이터 로딩/정제 함수 모듈 |
| `geocode_utils.py` | 📍 주소 → 좌표 변환 함수 모듈 |
| `map_generator.py` | 🗺️ folium 지도 생성 함수 모듈 |
| `전국휴지통표준데이터.csv` | 🚮 전국 휴지통 위치 데이터 |
| `12_04_01_E_공중화장실정보.xlsx` | 🚻 전국 공중화장실 위치 데이터 |

> 🔎 모든 코드는 프로젝트 루트에 위치합니다.

---

## 1️⃣ 프로젝트 개요

이 프로젝트는 **주소 기반으로 가까운 휴지통/공중화장실 위치를 지도에 시각화**하여,
사용자가 직관적으로 위치 정보를 탐색할 수 있도록 만든 Streamlit 기반 위치 서비스입니다.

---

## 2️⃣ 프로젝트 목적

- 🚮 **휴지통/공중화장실 위치를 지도에서 직관적으로 확인**
- 🗺️ **주소 입력만으로 주변 편의시설 탐색**
- 🧰 Streamlit 기반 **간단하고 직관적인 웹 UI** 제공

---

## 3️⃣ 설치 및 실행 방법

### 1) 저장소 클론
```bash
git clone https://github.com/david-han-0831/trash-map.git
cd trash-map
```

### 2) 가상환경 설정 (선택)

#### 🅰️ Python venv 사용
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

#### 🅱️ Anaconda 환경 사용
```bash
conda create -n trash-map python=3.10
conda activate trash-map
```

---

### 3) 필수 패키지 설치
```bash
pip install -r requirements.txt
```

---

### 4) Streamlit 앱 실행
```bash
streamlit run app.py
```
> 💡 실행 후 브라우저에서 안내된 주소로 접속하면 지도 서비스에 접속할 수 있습니다.

---

## 4️⃣ 주요 기능 소개

| 기능 | 설명 |
|------|------|
| 🚮 휴지통 지도 | 주소 입력 → 가까운 휴지통 3곳을 folium 지도에 표시 |
| 🚻 공중화장실 지도 | 주소 입력 → 가까운 공중화장실 3곳을 folium 지도에 표시 |
| 🗂️ 탭 UI | 상단 탭에서 휴지통/화장실 지도 전환 |
| 🌐 직관적 UI | Streamlit 기반의 반응형 웹 인터페이스 |

---

## 5️⃣ 데이터 설명

- `전국휴지통표준데이터.csv`
  - 🚮 전국 휴지통 위치 데이터 (UTF-8 인코딩)
  - 포함 정보: 위도, 경도, 설치장소명 등
- `12_04_01_E_공중화장실정보.xlsx`
  - 🚻 전국 공중화장실 위치 데이터 (엑셀)
  - 포함 정보: WGS84위도, WGS84경도, 소재지지번주소 등

---

## 6️⃣ 브랜치별 역할 및 내용

| 브랜치명 | 역할/내용 |
|----------|-----------|
| `student` | 학생이 직접 작성한 원본 코드 (보존용) |
| `refactor` | 기능은 동일하지만 구조적으로 개선된 리팩토링 코드 (함수화, 모듈화, 예외처리 등) |
| `master` | Streamlit 기반의 시각적 웹앱, 탭 UI 등 사용자 경험을 강화한 최신 버전 |

---

## 7️⃣ 참고 및 유의사항

- 데이터 파일은 반드시 프로젝트 루트에 위치해야 합니다.
- 지도 시각화는 folium + Streamlit 조합으로 구현되어 있습니다.
- Python 3.8 이상 환경에서의 실행을 권장합니다.
- requirements.txt에 명시된 패키지 외에 openpyxl, geopy, folium, streamlit 등이 필요합니다.
